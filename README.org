* Overview
This is my personal emacs config. I use emacs for the most time on
Windows 10. Currently I am using =GNU emacs 28.0.50= and there are no
major bugs, although on startup =cl= returns an error. If you are using
LSP, then you should use 27.X.XX. 

This config is maintained and will get updated in the future. There
could be some major changes with one commit, because of a really
boring lesson.

I use this config mostly for =org mode= but there are plans, that it is
going to be my IDE for everyhing, but that is delayed, because I have
some trouble with LSP.

* Installation (Windows)
An installation of =GNU emacs= preferably with version =28.0.50=. After
first start emacs will create a folder in the home directory (Windows:\\
=C:/users/USER/appdata/roaming/.emacs.d=). Then close emacs.

Next, install [[https://git-scm.com/download/win][git]] and open the file explorer in the emacs
directory. Then click the right mousbutton and select ="git bash
here"=. Then copy these commands:
1. =git init=
2. =git remote add origin https://www.github.com/domse007/.emacs.d=
3. =git pull origin master=

Now everytime there is a new commit, it can be installed with: 
=git pull origin master=

If this is just the startpoint, you could just make:


=git clone https://www.github.com/domse007/.emacs.d .=


in the =.emacs.d= directory. 

* dependencies
- autocomplete c headers wants to know where the header files are.

  - Windows: added to PATH Variables

  - For C++ development
- [[https://github.com/mlt/hunspell/releases/download/appveyor_v1.7.0/hunspell-msvc-Release-x64.zip][Hunspell installation]]
  - Windows:
    - add to =PATH=
    - new Variable =DICPATH=: =C:\...\.emacs.d\dict\=
- install Python and pip
  - download python language server
    - =pip install python-language-server[all]=
  - install additional functionality
    - =pip install pyls-black pyls-isort pyls-mypy=
- install Hack Font
- install [[https://rustup.rs/][Rust]]
  - install Rust linter [[https://github.com/rust-lang/rust-clippy][Clippy]]
    - environment variables are set automatically
  - change compiler with: =rustup set default-host
    x86_64-pc-windows-gnu=
  - install [[https://github.com/rust-analyzer/rust-analyzer/releases/tag][rust analyzer]]
    - either copy =rust-analyzer-windows.exe= to
      =C:\Users\USERNAME\.cargo\bin= or set additional environment
      variable.
    - rename file to =rust-analyzer.exe=
- install GNUplot
  - add it to =PATH=
    - checkbox available in installer

insertorg is part of my-macros. After calling the function with =C-c
m=. It can be exited with =C-g=. The function expect an input through the
minibuffer. Possible options are printed in the minibuffer. After that
press =RET= and then follow the instruction of the subfunctions.
Examples: =C-c m org RET RET RET= or =C-c m org RET test RET article
RET=. The functions are really basic and easy to modify.
