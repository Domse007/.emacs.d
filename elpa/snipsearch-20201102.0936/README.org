* About
Snipsearch is a major mode to organize and easily use your snippets.

* Install
Snipsearch can be installed with git, by cloning this repo. After
that, you have to put this into your config:

#+begin_src emacs-lisp
(require 'snipsearch)
#+end_src

Alternatively, you can use [[https://github.com/jwiegley/use-package][use-package]]:

#+begin_src emacs-lisp
(use-package snipsearch)
#+end_src
